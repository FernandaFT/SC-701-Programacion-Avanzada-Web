# SC-701 - Programación Avanzada Web

Proyecto desarrollado para el curso **SC-701 Programación Avanzada Web**, enfocado en el desarrollo de aplicaciones web modernas utilizando tecnologías frontend y backend.

---

## 📌 Descripción

Este repositorio contiene una aplicación web desarrollada con el objetivo de aplicar conceptos avanzados de programación web, incluyendo arquitectura MVC, conexión a bases de datos, manejo de formularios y operaciones CRUD.

El sistema permite administrar información mediante una interfaz dinámica, organizada y responsiva.

---

## 🛠️ Tecnologías Utilizadas

- PHP
- HTML5
- CSS3
- JavaScript
- Bootstrap
- MySQL
- Git & GitHub
